## 📖 О проекте

**Nano Banana Bot** — токсичный Telegram-бот без морали, фильтров и вежливости. Отвечает коротко, грубо, с матом. При этом умеет генерировать изображения и видео, анализировать видео покадрово и вести диалог с памятью.

## ✨ Фичи

- 🤬 **Токсичная личность**: общается максимально агрессивно — никаких извинений, никакой политкорректности, 1-3 предложения и в точку.
- 🎨 **Генерация изображений (`/image`)**:
  - Быстрая и качественная генерация картинок.
  - Поддержка генерации на основе исходного изображения (картинка в картинку) или альбома.
  - Пояснение ошибок (если запрос заблокирован фильтрами).
- 🎬 **Генерация видео (`/video`)**:
  - Мощная генерация видео (включая звук).
  - Можно прикрепить фото для создания видео-анимации из него.
  - Перевод запроса на английский если нужно.
- 🎞️ **Анализ видео и GIF**: разбивает видео на кадры + извлекает аудио, всё скармливается боту для детального анализа.
- 🎙 **Озвучка текста (`/tts`)**:
  - Генерация реалистичного голоса. Большой выбор голосов, несколько языков.
  - Поддерживает режиссёрские настройки: Сцена, Стиль, Темп и Акцент!
- 💾 **Долговременная память**: запоминает последние 100 сообщений диалога на каждый чат, не теряет при перезапуске. Понимает реплаи.
- 🔄 **Восстановление незавершённых генераций**: при перезапуске бот проверяет незавершённые задачи и досылает результат.
- 🛡️ **Защита от спама**: настраиваемые кулдауны на фото, видео и текст.
- 🚫 **Бан-лист**: встроенная система для блокировки пользователей.
- 💻 **Написание кода по запросу**: если прямо попросить — пришлёт профессиональный, рабочий код без обрезки и халтуры файлом.

## 🚀 Установка и запуск

### Требования
- Python 3.10+
- FFmpeg (`sudo apt install ffmpeg`) — для анализа видео
- PM2 (опционально, для работы 24/7)

### Шаг 1: Клонирование
```bash
git clone https://github.com/Rewixx-png/NanoBananaBot.git
cd NanoBananaBot
```

### Шаг 2: Зависимости
```bash
pip install aiogram aiohttp aiosqlite
```

### Шаг 3: Настройка ключей
Создайте `r.txt`. Формат — JSON.

### Шаг 4: Конфиг
Откройте `config.py` и укажите:
- `BOT_TOKEN` — токен от @BotFather
- `CHAT_ID` — ID чата для проверки подписки
- `TEXT_ONLY_CHAT_ID` — ID чата только с текстом
- `FULL_ACCESS_CHAT_ID` — ID чата с полным доступом

### Шаг 5: Запуск
```bash
# Разово
python3 main.py

# 24/7 через PM2
pm2 start main.py --name NanoBananaBot --interpreter python3
pm2 save
```

## 🛠️ Команды

| Команда | Описание |
|---|---|
| `/start` | Приветствие и проверка доступа |
| `/help` | Полный список возможностей |
| `/image [промпт]` | Генерация изображения |
| `/video [промпт]` | Генерация видео |
| `/tts [текст]` | Озвучка текста |
| `/clear` | Очистить историю диалога |

Тег бота или реплай на его сообщение → текстовый ответ.  
Отправить видео/GIF с тегом → покадровый анализ.

## 👑 Админ-команды
- `/all` — агрессивный созыв всех участников чата.
- `/stats` — статистика генераций.
- `/prompts` — логи промптов.
- `/limit` — установка кастомных лимитов.
- `/ban` и `/unban` — управление доступом.
- `/vip` — выдать безлимит на 24ч.
