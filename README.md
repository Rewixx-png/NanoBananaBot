## О проекте

**Nano Banana Bot** — Telegram-бот без морали, фильтров и вежливости. Отвечает коротко, грубо, с матом. При этом умеет генерировать изображения и видео, апскейлить картинки, озвучивать текст, анализировать видео покадрово, искать свежую инфу через Firecrawl, писать код и проекты файлами, и вести диалог с памятью.

Поддерживает два бота одновременно: основной (`BOT_TOKEN`) и второй (`BOT_TOKEN_2`) для режима дуэльного диалога, где два AI-персонажа базарят между собой.

## Фичи

**Токсичная личность** — общается максимально агрессивно, никаких извинений, 1-3 предложения и в точку.

**Генерация изображений** — несколько провайдеров: Gemini, GPT, FLUX через NVIDIA и Replicate для отдельных моделей. Можно прикрепить фото или альбом как основу. Если надо, бот сам сделает нормальный AI-промпт по твоей идее и фото. После генерации он автоматически пытается сделать апскейл 2x и прислать улучшенную версию без сжатия.

**Естественные запросы на рисование** — не обязательно писать `/image`. Если написать что-то вроде "нарисуй мне кота в космосе" или "сделай картинку с закатом", бот сам распознает запрос и нарисует. Умеет редактировать уже нарисованное — ответь реплаем на картинку с инструкцией типа "сделай его злым".

**Апскейл (`/up`)** — отправь картинку с командой `/up`, бот улучшит качество через AI upscaler и вернёт документом без сжатия.

**Генерация видео (`/video`)** — через Veo. Можно прикрепить фото для анимации из него. Поддерживает выбор модели Veo через inline-кнопки.

**Анализ видео и GIF** — разбивает видео на кадры, извлекает аудио, всё скармливается боту для детального анализа.

**Озвучка текста (`/tts`)** — Gemini TTS с настройкой голоса, стиля, сцены, темпа и акцента через inline-меню. Выбор модели через кнопки.

**Агрессивный роаст (`/bull`)** — реплаем на юзера запускает серию жёстких оскорблений в его адрес.

**Agentic web search через Firecrawl** — бот сам понимает, когда нужна свежая инфа. Не кидает сырой текст в поиск, а формулирует точные запросы, читает страницы, лезет по ссылкам до третьего уровня, гоняет цикл до 16 попыток и копит несколько релевантных источников перед ответом. Если достоверной инфы нет — честно скажет.

**Код и проекты** — попроси написать код или проект, бот отдаст один файл документом или полноценный проект в `.zip`. Перед отправкой проверяет Python/JSON/HTML синтаксис.

**Долговременная память** — запоминает последние 100 сообщений диалога на каждый чат, не теряет при перезапуске. Понимает реплаи.

**Восстановление незавершённых генераций** — при перезапуске бот проверяет незавершённые задачи и досылает результат.

**Лимиты и баны** — настраиваемые кулдауны, дневные лимиты генераций на чат, бан-лист.

**Дуэльный диалог (`/dual` / `/stopdual`)** — запускает два AI-персонажа (Нано и Банан), которые базарят между собой в чате. Второй бот работает через `BOT_TOKEN_2`.

## Установка и запуск

### Требования

- Python 3.10+
- FFmpeg (`sudo apt install ffmpeg`) — для анализа видео
- PM2 (опционально, для работы 24/7)

### Зависимости

```bash
pip install aiogram aiohttp aiosqlite python-dotenv
pip install pillow
```

Для запасного апскейла можно поставить `picwish`, но бот и без него попробует основной upscaler.

### Настройка токенов

Создайте `.env` в корне проекта:

```
BOT_TOKEN=токен_основного_бота
BOT_TOKEN_2=токен_второго_бота_для_dual
```

### Настройка API-ключей

Создайте `r.txt` в корне проекта. Формат — JSON:

```json
{
  "gemini": ["AIza...ключ1", "AIza...ключ2"],
  "openai": "sk-proj-...",
  "firecrawl": ["fc-..."],
  "nvidia": ["nvapi-..."],
  "openrouter": ["sk-or-..."],
  "replicate": ["r8_..."]
}
```

Ключи ротируются автоматически. Если ключ умирает — бот переключается на следующий.

Для некоторых провайдеров ключи также подтягиваются из `/root/RewTest/keyhunter.db`, если файл существует.

### Запуск

```bash
# Разово
python3 main.py

# 24/7 через PM2
pm2 start main.py --name NanoBananaBot --interpreter python3
pm2 save
```

## Команды

| Команда | Описание |
|---|---|
| `/start` | Приветствие и проверка доступа |
| `/help` | Справка по возможностям |
| `/clear` | Очистить историю диалога |
| `/image [промпт]` | Генерация изображения (Gemini / GPT / FLUX / Replicate) |
| `/up` | Апскейл картинки 2x через AI upscaler |
| `/video [промпт]` | Генерация видео через Veo |
| `/tts [текст]` | Озвучка текста с настройкой голоса и стиля |
| `/bull` | Реплаем на юзера — серия жёстких оскорблений |
| `/all` | Агрессивный созыв всех участников чата |
| `/stats` | Статистика генераций (только владелец) |
| `/prompts` | Логи промптов (только владелец) |
| `/limit [кол-во] [дней]` | Установить лимит генераций для чата |
| `/ban` | Забанить юзера (реплаем или ID) |
| `/unban` | Разбанить юзера |
| `/vip` | Выдать безлимит на 24 часа |
| `/dual` | Запустить дуэльный диалог двух ботов |
| `/stopdual` | Остановить дуэльный диалог |

**Естественные запросы:**
- Тегни бота или ответь реплаем на его сообщение — текстовый ответ.
- Напиши "нарисуй...", "сделай картинку..." — бот сам распознает и нарисует без `/image`.
- Ответь реплаем на нарисованную картинку с правкой — бот отредактирует.
- Напиши "найди в инете...", "поищи...", "что нового..." — agentic Firecrawl-поиск.
- Отправь видео/GIF с тегом — покадровый анализ.
- Попроси написать код или проект — получишь файл или `.zip`.

## Как работает интернет-поиск

1. Бот сам понимает, что нужна свежая инфа — не надо писать специальную команду.
2. Формулирует несколько точных Firecrawl-запросов с кавычками, минус-словами и source-операторами.
3. Ищет по разным источникам: web, Telegram, VK, X/Twitter, Instagram, TikTok, YouTube, Reddit, GitHub/GitLab, Habr/VC/DTF, Boosty, форумы.
4. Читает до 8 результатов за запрос, лезет по ссылкам до третьего уровня.
5. По найденным данным составляет новые запросы и гоняет цикл до 16 попыток.
6. Копит несколько релевантных источников, потом синтезирует ответ. Если нормальной инфы нет — честно скажет.
