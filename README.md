## О проекте

**Nano Banana Bot** — Telegram-бот без морали, фильтров и вежливости. Отвечает коротко, грубо, с матом. Под капотом — полноценный ReAct-агент с 21 инструментом, который умеет искать в интернете, генерировать медиа, выполнять код в изолированном Docker-контейнере, скачивать видео и многое другое.

Работает два бота одновременно: основной (`BOT_TOKEN`) и второй (`BOT_TOKEN_2`) для режима дуэльного диалога.

## Возможности

### 🤖 AI-агент (21 инструмент)

Бот автоматически определяет, нужен ли агент, и запускает ReAct-цикл (до 18 шагов). Агент сам формулирует план, вызывает инструменты, обрабатывает результаты.

**Поиск и данные**
- `web_search` — поиск через Firecrawl, до 6 результатов с markdown-контентом
- `scrape_url` — полный скрейпинг страницы (до 8000 символов)
- `fetch_json` — HTTP GET любого JSON API
- `calculate` — безопасный AST-калькулятор (все функции `math`, без `eval`)

**Медиа (отправляются прямо в чат во время работы агента)**
- `search_and_send_image` — ищет картинку через Firecrawl + ogImage, проверяет каждую через Gemini Vision, до 3 раундов с рефайном запроса
- `generate_image` — генерация через Gemini Imagen
- `download_image` — скачать фото по прямому URL
- `search_and_send_video` — ищет видео на YouTube/TikTok/VK, **верифицирует автора** (по URL или scrape страницы), скачивает через yt-dlp, до 3 раундов
- `download_video` — скачать видео по прямой ссылке (yt-dlp, до 720p, лимит 48 МБ)
- `text_to_speech` — синтез речи через Gemini TTS, голоса: Kore / Aoede / Charon / Fenrir / Puck

**Docker-сэндбокс** (изоляция: `--network=none`, `--memory=512m`, `--cpus=0.5`, `--user=sandbox`)
- `run_python` — Python в контейнере `hatani-sandbox`, доступно numpy/pandas/matplotlib/pillow/scipy/sympy
- `run_shell` — bash-команды в том же контейнере
- `write_file` / `read_file` — файлы в общем workspace, персистируют между шагами агента

**Утилиты**
- `qr_code` — QR-код → PNG в чат
- `create_chart` — bar/line/pie/scatter через matplotlib → PNG в чат
- `translate` — перевод текста на любой язык через Gemini
- `create_file` — создать текстовый/кодовый файл и отправить документом

**Выход**
- `generate_project` — собрать полноценный проект и прислать ZIP (с проверкой синтаксиса)
- `reply` — финальный текстовый ответ

**Примеры агентных запросов:**
```
"покажи загрузку CPU"        → run_shell('top -bn1')
"нагрузи оперативку"         → run_python(allocate list)
"найди картинку Standoff 2"  → search_and_send_image
"скачай эдит от kadzu vfx"   → search_and_send_video(creator='kadzu vfx')
"сделай график: Jan 100..."  → create_chart
"переведи на японский: ..."  → translate
"сколько места на диске"     → run_shell('df -h')
"сделай сайт про котов"      → web_search → generate_project
```

### 🎨 Генерация изображений (`/image`)

Мультипровайдерная генерация с выбором через inline-кнопки:

| Провайдер | Особенности |
|---|---|
| **Gemini** | Поддержка референсных фото и альбомов, auto-prompt через Gemini Pro |
| **GPT / OpenAI** | DALL-E 3 и gpt-image-2, редактирование по фото |
| **NVIDIA NIM / FLUX** | FLUX.1-schnell (быстро), FLUX.1-dev (качество), авто-перевод промпта |
| **OpenRouter** | Доступ к дополнительным моделям включая FLUX-варианты |
| **Replicate / NSFW** | FLUX-dev, Recraft v3, WAI-NSFW-Illustrious (аниме) с полным конфигуратором |

Дополнительно:
- Авто-улучшение промпта: если прикреплено фото, Gemini может сам составить детальный промпт
- Выбор температуры (0.1 — точно, 2.0 — хаос)
- Natural draw: напиши "нарисуй кота" без `/image` — бот сам поймёт
- Редактирование: реплай на сгенерированную картинку с инструкцией → бот отредактирует
- PIL-генерация: Gemini пишет Python-код → выполняется в sandbox → Gemini проверяет результат (до 5 попыток)

### 🎬 Генерация видео (`/video`)

- Провайдер: **Google Veo** (veo-2.0, veo-3.0 и другие — выбор через inline-кнопки)
- Длительность: 8 секунд, соотношение 16:9
- Поддержка фото как основы для анимации (Gemini описывает фото → Veo анимирует)
- Таймаут: 5 минут с прогресс-баром
- Восстановление при перезапуске бота

---

### 🎙 Озвучка (`/tts`)

- Провайдер: **Gemini TTS** (модель выбирается динамически)
- Голоса: Kore, Aoede, Charon, Fenrir, Puck и другие (с предпрослушкой)
- Настройки через inline-меню: сцена, стиль, темп, акцент
- Выход: `.ogg Opus` (voice-сообщение в Telegram)

| **Голосовые / аудио** | Транскрипция + ответ по существу |

### 🧠 Память и контекст

- **История чата**: SQLite, последние 100 сообщений, не теряется при перезапуске
- **Контекстный буфер**: последние 100 сообщений всех участников (включая не обращённые к боту) инжектируются как контекст
- **Защита от отравления**: при `PROHIBITED_CONTENT` буфер автоматически чистится, retry без контекста
- `/clear` — очищает оба: SQLite-историю и буфер в памяти

### 🌐 Agentic веб-поиск

Встроенный (не через агента) — срабатывает автоматически при обычных вопросах:

1. Gemini формулирует 8–12 точных Firecrawl-запросов
2. Ищет по: web, Telegram, VK, X, Instagram, TikTok, YouTube, Reddit, GitHub, Habr/VC/DTF, Boosty, форумы
3. До 8 страниц за запрос, углубляется по ссылкам до 3 уровней
4. До 16 итераций с рефайном по найденным данным
5. Синтезирует ответ из релевантных источников (порог качества 55/100)

## Установка и запуск

### Требования

- Python 3.10+
- Docker (для агентного сэндбокса)
- FFmpeg (`sudo apt install ffmpeg`)
- yt-dlp (`pip install yt-dlp`)
- PM2 (опционально, для 24/7)

### Зависимости Python

```bash
pip install aiogram aiohttp aiosqlite python-dotenv
pip install pillow qrcode matplotlib scipy sympy
pip install yt-dlp picwish
```

### Сборка Docker-сэндбокса

```bash
docker build -t hatani-sandbox:latest -f Dockerfile.sandbox .
```

Образ содержит: Python 3.11-slim + numpy, pandas, matplotlib, pillow, scipy, sympy.

### Переменные окружения (`.env`)

```env
BOT_TOKEN=токен_основного_бота
BOT_TOKEN_2=токен_второго_бота_для_dual
FIGMA_TOKEN=figd_ваш_figma_personal_access_token
FIRECRAWL_API_KEY=fc_ваш_ключ
```

### API-ключи (`r.txt`)

```json
{
  "gemini":    ["AIza...ключ1", "AIza...ключ2"],
  "openai":    "sk-proj-...",
  "firecrawl": ["fc-...ключ1", "fc-...ключ2"],
  "nvidia":    ["nvapi-..."],
  "openrouter":["sk-or-..."],
  "replicate": ["r8_..."]
}
```

Ключи ротируются автоматически. При 429 — ключ в кулдауне 65 сек; при 403 — 300 сек; при 401/402 — удаляется из файла. Дополнительный источник: `/root/RewTest/keyhunter.db` (если есть).

### Запуск

```bash
# Разово
python3 main.py

# 24/7 через PM2
pm2 start main.py --name NanoBananaBot --interpreter python3
pm2 save
```

## Команды

| Команда | Описание | Доступ |
|---|---|---|
| `/start` | Приветствие и проверка доступа | Все (личка) |
| `/help` | Справка по возможностям | Все |
| `/clear` | Очистить историю + контекстный буфер | Все члены |
| `/image [промпт]` | Генерация изображения | Все; cooldown 30 сек |
| `/up` | Апскейл картинки 2x без сжатия | Все члены |
| `/video [промпт]` | Генерация видео через Veo | Все члены; cooldown 60 сек |
| `/tts [текст]` | Озвучка с настройкой голоса и стиля | Все; cooldown 30 сек |
| `/bull` | Роаст реплаем на юзера | Все |
| `/all` | Тегнуть всех участников чата | Все (группы) |
| `/figma [описание]` | Создать дизайн в Figma через AI | Все |
| `/dual` | Запустить диалог двух AI-ботов | FULL_ACCESS_CHAT |
| `/stopdual` | Остановить dual-режим | FULL_ACCESS_CHAT |
| `/stats` | Статистика генераций | Только владелец |
| `/prompts [user_id]` | Логи промптов | Только владелец |

| `/ban` | Забанить юзера (реплай или ID) | Только владелец |
| `/unban` | Разбанить юзера | Только владелец |
| `/vip` | Безлимит на 24 часа | Владелец + ALLOWED |

**Естественные запросы (без команды):**
- Тегни или ответь реплаем → текстовый ответ с памятью
- "нарисуй...", "сгенерируй картинку..." → автодетект → рисует
- "найди видео от [автор]", "скачай эдит..." → агент → поиск + верификация + скачивание
- "найди фотку [тема]" → агент → поиск с Gemini Vision-проверкой
- "нагрузи проц", "чекни whoami" → агент → Docker sandbox
- "построй график", "переведи на яп", "сделай QR" → агент → соответствующий инструмент
- "сделай сайт/программу/бота" → агент → генерация проекта ZIP
- "найди в инете...", "что нового у..." → agentic Firecrawl-поиск
- Отправь видео/GIF → покадровый анализ + аудио
- Отправь голосовое → транскрипция + ответ
- Отправь документ/zip → анализ файла, отвечает на вопросы

## Архитектура

```
main.py          — точка входа, 2 бота, BanMiddleware, DualHistoryMiddleware
config.py        — константы, токены из env, SYSTEM_PROMPT
handlers.py      — все команды и обработчики (~2800 строк)
ai_services.py   — AI-вызовы: Gemini/GPT/NVIDIA/Replicate/Veo/TTS/Firecrawl
agent.py         — ReAct-агент с 21 инструментом и Docker-сэндбоксом
state.py         — in-memory state (cooldowns, pending requests, context buffers)
database.py      — aiosqlite: история, статистика, баны, лимиты, логи промптов
keys_manager.py  — ротация ключей из r.txt + keyhunter.db
dual_bot.py      — второй бот (Банан/Нано), дуэльный AI-диалог
figma_bridge.py  — HTTP-сервер на порту 7432 для Figma Plugin
esrgan_model.py  — ESRGAN (anime.pth) — запасной апскейлер
Dockerfile.sandbox — Python 3.11-slim образ для агентного сэндбокса
```
